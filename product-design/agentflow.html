The Challenge

Democratising Agentic AI

Building AI agents required writing hundreds of lines of Python. Product managers, ops teams, and domain experts had powerful ideas — but zero access to agentic AI without engineering help. The gap between idea and implementation was costing weeks.

😤 Problems Identified

  • Non-technical users fully excluded from building agents
  • Existing tools (LangSmith, Flowise) designed only for engineers
  • No live debugging — impossible to see why an agent failed
  • Deployment required DevOps expertise and CLI knowledge
  • No version history — agent configs lost on every iteration

✅ Design Solutions

  • Node-based drag-and-drop canvas with natural language node names
  • Dual mode: "Builder" for PMs, "Code" for engineers — same graph
  • Live execution trace panel — see each node's input/output in real-time
  • One-click deploy with auto-scaling infra hidden behind a button
  • Git-style version history with diff view for agent graphs
